cloudbees opensource community

Welcome to CloudBees on github

At CloudBees we hand craft only the finest open source, artisanally (is that even a word?). This github group contains a lot of our open source - some are functional parts of the system for you to fork and use - others are libraries/utilities that we use and hope you find useful. Send us a pull request !

See us at http://developer.cloudbees.com

Jenkins

We are big supporters of Jenkins - this project is hosted over here: http://jenkins-ci.org/ for all your viewing pleasure. Some plugins you may find in this github group.

If you have an open source project you want build via Jenkins for free - please to try out https://buildhive.cloudbees.com - all your need is your github account.

ClickStarts you will find here

ClickStarts are pre-built stacks that people can build from - generally they are repos that end with 

-clickstart

So you can easily spot them. If you want to make your own - fork one that is similar - and go from there. Send us a pull request if you like.

ClickStacks

ClickStacks are runtime containers that are used by cloudbees - these are forkable - each one contains instructions on how to use if you want to customise it to your own needs. These typically end with 

-clickstack

so you can easily spot them - crazy huh !

Use your favourite git or github client and enjoy !
