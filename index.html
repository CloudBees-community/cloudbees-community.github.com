Welcome to CloudBees on github

At CloudBees we handcraft only the finest open source, artisanally (is that even a word?).

This github group contains the open source bits that make up the CloudBees platform - many are core parts of the system for you to fork and use - others are libraries/utilities that we use and hope you find useful. Send us a pull request !

See us at http://developer.cloudbees.com

The rest?

A hodge podge of utilities and important tools. And perhaps some bad ideas. No, all good ones, I just checked.

There are still some projects located at https://github.com/cloudbees as well.
