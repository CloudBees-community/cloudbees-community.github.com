cloudbees opensource community

Welcome to CloudBees on github

At CloudBees we hand craft only the finest open source, artisanally (is that even a word?). This GitHub group contains a lot of our open source projects. Some are functional parts of the system for you to fork and use; others are libraries/utilities that we use and hope you find useful. Send us a pull request!

See us at developer.cloudbees.com

Jenkins

We are big supporters of Jenkins. This project is hosted at jenkins-ci.org for all your viewing pleasure. Some plugins may be found in this GitHub group.

If you have an open-source project you want build via Jenkins for free, please try out buildhive.cloudbees.com—all your need is your GitHub account.

ClickStarts you will find here

ClickStarts are pre-built stacks that people can build from. Generally they are repos that end with -clickstart so you can easily spot them. If you want to make your own, fork one that is similar and go from there. Send us a pull request if you like.

ClickStacks

ClickStacks are runtime containers that are used by CloudBees. These are forkable—each one contains usage instructions if you want to customise it to your own needs. These typically end with -clickstack so you can easily spot them—crazy huh!

Use your favourite Git or GitHub client and enjoy!
