Welcome to CloudBees on github

At CloudBees we handcraft only the finest open source, artisanally (is that even a word?).

This github group contains the open source bits that make up the CloudBees platform - many are core parts of the system for you to fork and use - others are libraries/utilities that we use and hope you find useful. Send us a pull request !

See us at http://developer.cloudbees.com

JENKINS CI

We are big supporters of Jenkins - this project is hosted over here: https://jenkins-ci.org/ for all your viewing pleasure. Some plugins you may find in this github group. We support many FOSS projects by providing free build infrastructure - if you have an interesting project - let us know so we can help!

If you have an open source project in github and you want to build via Jenkins for free - please to try out https://buildhive.cloudbees.com - all your need is your github account.
